# KOFileUnpacker

KOFileUnpacker is a C++ and MFC-based utility designed to read and analyze Knight Online `.HDR` files and unpack corresponding `.SRC` data files based on the parsed structure.

This project was created for educational and research purposes to understand legacy file structures used in Knight Online.

---

## 📌 Features

- Reads Knight Online `.HDR` archive files
- Parses file metadata such as:
  - File names
  - Offsets
  - Sizes
- Extracts and unpacks data from `.SRC` files based on HDR mapping
- Simple MFC-based GUI for file operations
- Lightweight and fast execution

---

## ⚙️ Requirements

To build and run this project, you need:

- Visual Studio 2022 (or later)
- C++20 compiler support
- Windows SDK
- MFC (Microsoft Foundation Classes) support installed
- Windows 10/11 (recommended)

Make sure **MFC support is enabled** during Visual Studio installation:
> Individual Components → “C++ MFC for latest v143 build tools”

---

## 🛠️ Build Instructions

1. Open the `.sln` file in Visual Studio
2. Select configuration:
   - Debug / Release
   - x86 or x64 (depending on your setup)
3. Build solution
4. Run the application
